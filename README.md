# Examples for the seminar on C++20 coroutine feature
The following examples are materials for the presentation on the C++20 coroutine feature shown during the seminar by Virtual Reality student science group from Warsaw University of Technology. 

The examples include:
- Fibonacci sequence generator,
- Binary tree iterator,
- Coroutine as a wrapper around an asynchronous worker to clean up after it completes,
- State machine for the use in a Markov Chain.
